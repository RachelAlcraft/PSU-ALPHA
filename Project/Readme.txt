Project info for Msc Project

To reproduce list of pdb files for high res list
1) Download files from : https://www.rcsb.org/pages/general/summaries
2) Run program: MSC-HighResList
3) Run R script to download pdb files: DownloadPDBFiles.R
4) Run program MSC-AnnotateHighResList to add information to HighResList from the downloaded pdb files
